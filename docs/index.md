### 功能
1. 在**任何应用全屏状态顶端显示**.(支持鼠标穿透).
2. **时钟**, 可选是否置顶, 12/24小时制。
3. **倒计时**, 自定义时长. 结束提示音.
4. **显示网速**.
5. 自定义大小, 字体颜色.

### 开机自启动
* **系统偏好设置>>用户与群组>>登录项** 在其中添加**zClock**即可
